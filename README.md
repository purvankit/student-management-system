# student-management-system
Design and implementation of RESTful APIs using Node.js and Express that read and process JSON data from requests or files.
# Objective:
  • To understand the fundamentals of RESTful API architecture.
  • To create REST endpoints that accept and return data in JSON format.
  • To parse incoming JSON data using Express middleware (express.json()).
  • To simulate basic CRUD operations using in-memory data or a static JSON file.
# Platform: 
Node.js, Express.js, Postman, JSON file for data persistence, Visual Studio Code, npm/yarn
